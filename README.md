# Genshin Impact Player Engagement Analytics

## Live Analytics Website
Explore the interactive dashboard:

https://genshin-impact-chi.vercel.app/

## Project Overview
This project analyzes player engagement and gameplay behavior in Genshin Impact using survey data collected from active players.

The goal of the project was to understand player preferences, engagement patterns, and gameplay trends through primary survey data.

## Dataset
Survey data collected from Genshin Impact players.

Key statistics:
- 63 responses
- 152 survey views
- 41.45% response rate
- Average completion time: 7 minutes

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Data Visualization
- Survey Research
